Rawg Gamer App is a native iOS gaming discovery app built with SwiftUI. It connects to the RAWG.io API to serve a library of 500,000+ games with rich metadata, reviews, and screenshots.

Technical Highlights

  • SwiftUI with Combine for reactive data binding
  • Clean Architecture: separate Data, Domain, and Presentation layers
  • Core Data for offline caching and favourites
  • Firebase Analytics + Crashlytics for production monitoring
  • Sentry for crash reporting with detailed breadcrumbs
  • Full open-source codebase with UI tests
