DefiFundr is a decentralized crowdfunding platform that uses Ethereum smart contracts to ensure campaigns are transparent and trustless. Backers interact directly with on-chain contracts — no intermediaries.

## Technical Highlights

- Solidity smart contracts audited for common vulnerabilities
- Go backend with PostgreSQL + SQLC for off-chain indexing
- Flutter mobile app with Go-Ethereum (geth) integration for wallet management
- Full open-source codebase under MIT license
- End-to-end encryption for campaign creator data
