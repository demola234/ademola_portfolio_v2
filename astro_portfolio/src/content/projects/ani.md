Ani is a super-app for anime, manga, and gaming enthusiasts. Built with Flutter, it provides an extensive content library and community-driven features with a clean, performant mobile experience.

## Technical Highlights

- Clean Architecture with feature-based folder structure
- BLoC for predictable state management across all screens
- Firebase for auth, Firestore for real-time data, Crashlytics for stability monitoring
- Sentry integration for error tracking in production
- Multi-flavor setup for dev / staging / production environments
