TellerOne is a fintech platform for managing finances across multiple currencies and geographies. It supports real-time cross-border transactions, multi-wallet management, and currency conversion.

## Technical Highlights

- BLoC for complex financial state flows
- Firebase Auth with biometric login support
- Automated CI/CD pipeline with GitHub Actions
- Clean Architecture ensuring business logic is UI-independent
- Comprehensive test coverage including golden tests
