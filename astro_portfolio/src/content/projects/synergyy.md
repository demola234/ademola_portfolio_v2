Synergyy is a talent discovery and collaboration platform focused on Africa's next generation. It connects talented individuals with opportunities through a community-first approach.

## Technical Highlights

- WebSocket for real-time chat and collaboration features
- Firebase Cloud Messaging for push notifications
- Fastlane + GitHub Actions for automated deployments to App Store and Play Store
- BLoC for managing complex real-time state
- Clean Architecture with domain-driven design
